# Word Meaning
1 HTML HTML stands for Hypertext Markup Language. It is used to create the basic structure of a webpage
2 Debugging Debugging is the process of identifying and removing errors in written code
3 Hard-coded numbers Hard-coded number is a term used to designate literal numbers (not calculted or referenced amounts such as variables or dependent on variables for example)
3 HTTP HTTP stands for HyperText Transfer Protocol, it is the underlying protocol used by the World Wide Web.
3 Refactoring Code refactoring is the process of improving existing code through restructuring, editing and cleaning up while maintaining the exact same behaviour.
